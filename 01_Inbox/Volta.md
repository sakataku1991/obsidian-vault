## 概要
- 「Node.js」のバージョン管理ツール

## 初期設定
1. ターミナルで以下のコマンドを実行し、`Volta` をインストールする。

```Shell
brew install volta
```

2. バージョンを確認する。

```Shell
volta -v
```

3. パスを確認する。

```Shell
which volta
```

4. 以下のようなパスが表示されればOK（Intel Mac の場合）。

`/usr/local/bin/volta`

## 備考
ちなみに `Volta` で管理しているパッケージを一覧表示して確認するコマンドは以下！

```Shell
volta list
```
